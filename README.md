# FairwaysIQ App

FairwaysIQ is a golf GPS and club recommendation app prototype.

This repository tracks two implementations:

- `FairwayIQExpo/`: Windows-friendly Expo React Native app for testing on iPhone with Expo Go.
- `FairwayIQ-iOS/`: native SwiftUI/Xcode prototype for future native iOS development.

The current active build path is the Expo app.
